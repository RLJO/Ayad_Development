Odoo 12 All in One Financial Reports in PDF and EXCEL

Missing All Accounting reports on Odoo12 Community Edition? Don't worry, With this app we make it available.All Financial PDF Reports deprecated in Odoo Version 12.0, This report available on earlier version of odoo until Odoov11. For odoov12 We have developed this app to provides same Report as it was providing in its earlier version additionally we provides excel option on same report. You can Print Balance Sheet Report,Print Profit Loss Report, Print Trial Balance Report,Print General Ledger Report, Print Partner Ledger Report, Print Aged Partner Report,Print Account Tax and Tax Summary Report, Print Journal audit report in Excel and PDF both format using all the filters i.e Date Range filter, Journal filter, Posted/Unposted entry etc. We have also developed the other deprecated modules of Odoo for v12, If you interested please check our other Apps.

Features

Print All Accounting reports.

All Odoo12 missing accounting report are available in this app.

Print Reports in Excel and PDF format.

Print all important accounting reports in both Excel and PDF format.

Print Accounting Tax reports.

Print Account Tax reports in PDF and Excel format.

Print Journal Audit report.

Print Journal Audit reports in PDF and Excel format.

Different Partner Reports.

Print Aged Partner Report and Partner Ledger Report in PDF and Excel format.

Financial Accounting Report.

Print Financial accounting reports like Balance sheet, Profit Loss, Trial Balance, General Ledger in PDF and excel format.

Multiple Filter and Sort by Option.

Date Range filter, Posted/All entries filter and sort by option with Date and journal entry number.

Journal Filter on Report.

User can print report with multiple selection of journals.
