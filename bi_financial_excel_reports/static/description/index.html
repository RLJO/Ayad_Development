Odoo12 All Financial Reports in Excel and PDF

Missing Accounting reports on Odoo12 Community Edition? Don't worry, With this app we make it available.Financial PDF Reports deprecated in Odoo Version 12.0, This report available on earlier version of odoo until Odoov11. For odoov12 We have developed this app to provides same Report as it was providing in its earlier version additionally we provides excel option on same report. You can Print Balance Sheet Report,Print Profit Loss Report, Print Trial Balance Report,Print General Ledger Report in Excel and PDF both format using all the filters i.e Date Range filter, Journal filter, Posted/Unposted entry etc. We have also developed the other deprecated reports and modules of Odoo for v12, If you interested please check our other Apps

Features

Available Account Report.

Balance sheet, Profit Loss, Trial Balance, General Ledger.

Date Filter Option.

Provide Date Range filter on wizard for all reports.

Print Report In Excel Format.

All reports have a option to print in Excel so user can print reports in PDF as well as in Excel format.

Other Options.

Provide other options on wizard as a Enable Comparison, Column Label, Print, Display Debit/Credit column, Display Accounts etc.
