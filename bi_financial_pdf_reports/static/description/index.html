Odoo12 Financial Reports in PDF

Missing Accounting reports on Odoo12 Community Edition? Don't worry, With this app we make it available.Financial PDF Reports deprecated in Odoo Version 12.0, This report available on earlier version of odoo until Odoov11. For odoov12 We have developed this app to provides same Report as it was providing in its earlier version. You can Print Balance Sheet Report,Print Profit Loss Report, Print Trial Balance Report,Print General Ledger Report in PDF format using all the filters i.e Date Range filter, Journal filter, Posted/Unposted entry etc. We have also developed the other deprecated reports and modules of Odoo for v12, If you interested please check our other Apps.

Features

Available Account Report.

Balance sheet, Profit Loss, Trial Balance, General Ledger.

Date Filter Option.

Provide Date Range filter on wizard for all reports.

Different Filter Options.

Provide Filter By option with different fields as well as Posted/All entries filter available on wizard.

Other Options.

Provide other options on wizard as a Enable Comparison, Column Label, Print, Display Debit/Credit column, Display Accounts etc.
