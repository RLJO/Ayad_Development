Odoo 12 Journals Audit Reports in PDF and Excel

Missing Accounting reports on Odoo12 Community Edition? Don't worry, With this app we make it available.Journal Audit Report deprecated in Odoo Version 12.0, This report available on earlier version of odoo until Odoov11. For odoov12 We have developed this app to provides same Report as it was providing in its earlier version additionally we provides excel option on same report. You can Print Journals Audit Reports in Excel and PDF both format using all the filters i.e Date Range filter, Journal filter, Posted/Unposted entry etc. We have also developed the other deprecated reports and modules of Odoo for v12, If you interested please check our other Apps.

Features

Excel Report Option.

Print Journals audit reports in Excel Format.

PDF report Option.

Print Journals audit reports in PDF format.

Journal Filter on Report.

Print Different Journal's Report using Journal Filter, User can select multiple journals also and print selected journals reports in group.

Date and Other Filter Option.

Date Range filter as well as Posted/All entries filter available on wizard.It also have sort by option with Date and journal entry number.

Journals Audit Report Menu

Figure shows the PDF Reports Option under Invoicing > Reporting Menu.


Journals audit report sorted by date

In wizard select target moves and sort by date.

You can see journals audit report sort by date in pdf format.

When you click on Print Excel button new wizard will open, you can download your file from here.

You can see excel report with target move and sort by date.

Journals audit report sorted by journal entry number

In wizard select target move and sort by journal entry number.

You can see journals audit report in pdf format sort by journal entry number.

You can see excel report with target move and sort by journal entry number.

Journals audit report between start date and end date

In wizard select start date and end date to print journal report.

You can see journals audit report between start date and end date in pdf format.

You can see excel report between start date and end date.

Journals audit report sorted by journals

In wizard select journal items that you wants to print.

You can see journals audit report with journal items in pdf format.

You can see excel report with journal items.
